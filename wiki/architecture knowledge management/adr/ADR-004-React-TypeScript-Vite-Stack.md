# ADR-004: React + TypeScript + Vite Stack

## Status
Accepted

## Date
2026-02-08

## Context
The UI must be fast, maintainable, and accessible, with a productive developer experience.

## Decision
Use React with TypeScript for UI and state management, and Vite for build tooling.

## Consequences
- Strong typing and maintainability.
- Fast dev server and modern bundling.
- Aligns with existing component ecosystem.
